# Support

nicechunk-worldgen is a focused NiceChunk repository generated from the main working tree.

## Questions and Bugs

Use GitHub issues for reproducible bugs, focused feature requests, and public documentation problems. Include the repository name, commit hash, reproduction steps, expected behavior, actual behavior, and validation output.

## Security and Private Operations

Do not post private keys, tokens, passwords, server IPs, deployment scripts, or machine-specific configuration in public issues. Send suspected vulnerabilities or leaks privately to the project owner and rotate exposed credentials before public discussion.

## Repository Scope

If an issue spans multiple NiceChunk repositories, open it in the repository closest to the failing surface and link related repositories in the description.
